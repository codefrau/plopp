Welcome to PLOPP, the cool 3D painting tool!

Plopp is running in Demo mode right now. In Demo mode not all Plopp features are available. To enable all the great Plopp features you need to activate Plopp with a serial number which you can purchase online.

The same serial number is valid for all Plopp versions on this site. You can download and use one or more of the themes available with the same serial number. You need one serial number per machine you would like to run Plopp on.

In the full version all painted figures, backgrounds, and scenes are saved automatically. In Demo mode, however, this feature is disabled so nothing is saved when you exit Plopp.
